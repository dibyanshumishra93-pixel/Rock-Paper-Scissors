This is a simple Rock-Paper-Scissors game written in C++ that runs in the terminal.


HOW IT WORKS:-
You will be given three choice as the following:
1.'r' for rock
2.'p' for paper
3.'s' for scissors
Then you will be asked to enter any of the following. You are requested to only enter 'r', 'p' or 's' otherwise your input may be considered invalid.
The computer then generates a random action between the three without knowing your choice.
Then you will be shown if you win, lose or tie against the computer.

HOW TO RUN IT:-
METHOD 1 - Using g++ (Command Line) -

1. Make sure you have a C++ compiler installed (like [MinGW](https://www.mingw-w64.org/) on Windows, or `g++` on Linux/Mac).
2. Open a terminal in the project folder.
3. Compile the program:
   g++ rockpaperscissorsgame.cpp -o rps
4. Run the compiled program:
   - On Windows:
   rps.exe
   - On Linux/Mac:
     ./rps
     
METHOD 2 - Using an IDE (e.g. VS Code, Code::Blocks, CLion) - 
1. Open the project folder in your IDE of choice.
2. Make sure a C++ compiler (like MinGW or GCC) is installed and configured in the IDE.
3. Open `rockpaperscissorsgame.cpp`.
4. Click **Run** / **Build and Run** (or use the shortcut, e.g. `Ctrl+F5` in VS Code with the C++ extension).
5. The game will launch in the integrated terminal or a separate console window.
